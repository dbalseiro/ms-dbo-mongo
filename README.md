# ms-dbo-mongo

REST API for dbo database

## Pre Requisites

* Install [stack]
* Install [dotenv]
* Install [mongoDB]

## Setup and installation

* clone this repo and build the project

```
$ git clone
$ cd ms-dbo-mongo
$ stack build
```
